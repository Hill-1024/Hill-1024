いろいろな分野を学びながら、思いついたものを実際に動くプロジェクトにするのが好きです。

## 関心分野

- フロントエンドとデスクトップアプリ：React、Astro、Vue、Tauri、Electron。
- Android WebView / Capacitor アプリ開発。
- AI アプリケーション、Agent ワークフロー、ツール呼び出し。
- 個人サーバー、プロキシプロトコル、デプロイスクリプト、WebUI。
- Markdown 執筆、ブログテーマ、コンテンツ管理。
- C/C++、授業課題、少し低レイヤー寄りのエンジニアリング練習。

銀の弾丸は信じていません。合っていて、明確で、保守しやすい選択は、見た目だけ高度な選択より大切なことが多いです。

## 中文

你好，我是 Hill。

我喜欢把想法做成可以运行、可以部署、可以继续维护的项目：从 WebUI、桌面应用、Android WebView / Capacitor，到 AI 工具、个人服务器、Markdown 写作和博客系统。

我不太相信“银弹”。技术选择最好回到具体问题：边界是否清楚，运行是否稳定，用户是否真的能用，下一次维护的人是否还能看懂。

## English

Hi, I am Hill.

I build practical projects that can run, ship, and keep evolving: Web UIs, desktop apps, Android WebView / Capacitor apps, AI tooling, personal infrastructure, Markdown writing tools, and blog systems.

I do not treat any stack as a silver bullet. The better choice is usually the one with clear boundaries, real usability, stable operation, and maintainable code after the first launch.

## 日本語

こんにちは、Hill です。

動くだけでなく、配布できて、運用しながら育てられるものを作るのが好きです。Web UI、デスクトップアプリ、Android WebView / Capacitor、AI ツール、個人サーバー、Markdown エディタ、ブログシステムなどを触っています。

どの技術も万能薬とは考えていません。境界が明確で、実際に使えて、安定して動き、後から読んでも直せることを大切にしています。
